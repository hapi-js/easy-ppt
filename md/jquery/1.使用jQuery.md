# 使用 `jQuery`

---

## 什么是 `jQuery`

1. jQuery是一个快速，小型且功能丰富的JavaScript库（框架）。
2. 借助易于使用的API（方法），使HTML文档的遍历和操作，事件处理，动画和Ajax等事情变得更加简单。
3. 结合了多功能性和可扩展性（插件）。

===

## 下载 `jQuery`

下载地址

```bash
https://jquery.com/
```

===

## 引入 `jQuery`

引入代码

```html
<script src="./jquery-3.5.1.min.js"></script>
```
